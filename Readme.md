Case completed february 2023 for PFA by Anton Baht.

## Run project
Clone the repo:
git clone git@github.com:bahtman/pfa_case.git
Install all packages:
pip install -r requirements.txt
Activate virtual environment:
.\env\Scripts\Activate.ps1
